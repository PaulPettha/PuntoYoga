
# PuntoYoga – Progressive Web App

Sito per amanti dello yoga. Include eventi, mappa, profili, bacheca.  
Pubblica le pagine su GitHub Pages e visita:  
`https://<tuo-username>.github.io/puntoyoga`

## Contenuto

- `index.html` – homepage
- `eventi.html` – eventi community
- `mappa.html` – mappa luoghi yoga
- `profili.html` – utenti e preferenze
- `bacheca.html` – post ispirazionali
- `manifest.json` – installazione app
- `service-worker.js` – offline support

Installabile su smartphone come app!
